Introduction

This document outlines the BinaryFile Schema created by the Open Microscopy Environment Group. Detailed information on the schema and future development plans are available at http://www.openmicroscopy.org/site/support/file-formats. Further information on the groups work is available at http://www.openmicroscopy.org/

Status

This schema is current and at version 1.

A list of the current versions of all the Open Microscopy Environment Group schemas is available at http://www.openmicroscopy.org/Schemas/

Schema

The schema XSD file is BinaryFile.xsd.

Note: Some browsers will try to render XSD files when you view them. This can result in either a blank screen or unformatted text. Choose to either download the file or view the source.
