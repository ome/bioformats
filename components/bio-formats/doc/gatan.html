File format name: Gatan Digital Micrograph 3
Maintainer: Gatan
Software that produces this:Gatan Digital Micrograph
Other software that reads this format:Gatan Digital Micrograph, DM3_Reader ImageJ plugin, EMAN
Specifications: Nothing official; see here.
Structure:
See http://rsb.info.nih.gov/ij/plugins/DM3Format.gj.html
This format's structure is not always consistent, so there are probably some older or newer files that aren't opened properly. Also, there is an older Gatan format, Digital Micrograph 2, that is similar, but not supported. If you have files in this format, please send them to us and we will add support.
