Bio-Rad PIC
===========

Extensions: .pic, .raw, .xml

Developer: Bio-Rad

Owner: `Carl Zeiss, Inc. <http://www.zeiss.com/>`_

Support
-------

SCIFIO: |no|

Export: |no|

Supported Metadata Fields: `Bio-Rad
Reader <http://dev.loci.wisc.edu/trac/software/wiki/BioRadReader>`_

Freely Available Software: `Bio-Rad PIC reader plugin for
ImageJ <http://rsb.info.nih.gov/ij/plugins/biorad.html>`_

We currently have: 

-  a PIC specification document (v4.5, in PDF)
-  an older PIC specification document (v4.2, from 1996 Dec 16, in DOC)
-  a large number of PIC datasets
-  the ability to produce new datasets
