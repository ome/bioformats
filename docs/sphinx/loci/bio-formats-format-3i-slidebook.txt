3i SlideBook
============

Extensions: .sld

Developer:  `Intelligent Imaging Innovations`_

Owner: `Intelligent Imaging Innovations`_

Support
-------

SCIFIO: |no|

Export: |no|

Officially Supported Versions:  4.1, 4.2

Supported Metadata Fields: `Slidebook Reader <http://dev.loci.wisc.edu/trac/java/wiki/SlidebookReader>`_

We currently have: 

-  Numerous SlideBook datasets
-  Some :source:`Bio-Formats developer
   notes <components/bio-formats/doc/slidebook.html>`
   for SlideBook

We'd like to have: 

-  A SlideBook specification document
-  More SlideBook datasets (preferably acquired with the most recent
   SlideBook software)

Additional Information
----------------------

Source Code: :source:`Slidebook Reader <components/bio-formats/src/loci/formats/in/SlidebookReader.java>`

Notes: 

We strongly encourage users to export their .sld files to OME-TIFF using
the Slidebook software.  Bio-Formats is not likely to support the full
range of metadata that is included in .sld files, and so exporting to
OME-TIFF from Slidebook is the best way to ensure that all metadata is
preserved.
