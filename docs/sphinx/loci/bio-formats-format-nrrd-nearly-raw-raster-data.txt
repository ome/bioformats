NRRD (Nearly Raw Raster Data)
=============================

Extensions: .nrrd, .nhdr, .raw, .txt

Developer: `Teem developers <http://teem.sourceforge.net/>`_

Support
-------

SCIFIO: |yes|

Export: |no|

Supported Metadata Fields: `NRRD Reader <http://dev.loci.wisc.edu/trac/software/wiki/NRRDReader>`_

Freely Available Software: `nrrd (NRRD reference library) <http://teem.sourceforge.net/nrrd/>`_

Sample Datasets: `Diffusion tensor MRI
datasets <http://www.sci.utah.edu/%7Egk/DTI-data/>`_

We currently have: 

-  A `nrrd specification
   document <http://teem.sourceforge.net/nrrd/format.html>`_ (v1.9, from
   2005 Dec 24, in HTML)
-  A few nrrd datasets
