DICOM
=====

Extensions: .dcm, .dicom

Developer: `National Electrical Manufacturers Association <http://www.nema.org/>`_

Support
-------

SCIFIO: |yes|

Export: |no| 

Supported Metadata Fields: `DICOM Reader <http://dev.loci.wisc.edu/trac/software/wiki/DicomReader>`_

Freely Available Software:  `OsiriX Medical Imaging Software <http://www.osirix-viewer.com/>`_

Freely Available Software: `ezDICOM <http://www.sph.sc.edu/comd/rorden/ezdicom.html>`_

Freely Available Software: `Chris Rorden's list of free DICOM
readers <http://www.sph.sc.edu/comd/rorden/dicom.html#links>`_

Sample Datasets:  `MRI Chest from FreeVol-3D web
site <http://members.tripod.com/%7Eclunis_immensus/free3d/hk-40.zip>`_

Sample Datasets: `Medical Image Samples from Sebastien Barre's Medical Imaging
page <http://www.barre.nom.fr/medical/samples/>`_

Sample Datasets: `DICOM sample image sets from OsiriX web
site <http://osirix-viewer.com/datasets/>`_

We currently have: 

-  `DICOM specification
   documents <http://medical.nema.org/dicom/2007/>`_ (PS 3 - 2007, from
   2006 Dec 28, in DOC and PDF)
-  Numerous DICOM datasets
