Olympus FluoView TIFF
=====================

Extensions: .tif

Owner: `Olympus <http://www.olympus.com/>`_

Support
-------

SCIFIO: |no|

Export: |no|

Supported Metadata Fields: `FluoView Reader <http://dev.loci.wisc.edu/trac/software/wiki/FluoviewReader>`_

Freely Available Software: `DIMIN <http://www.dimin.net/>`_

We currently have: 

-  A FluoView specification document (from 2002 Nov 14, in DOC)
-  Olympus's FluoView Image File Reference Suite (from 2002 Mar 1, in
   DOC)
-  Several FluoView datasets
