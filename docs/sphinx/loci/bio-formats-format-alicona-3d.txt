Alicona 3D
==========

Extensions:  .al3d

Owner:  `Alicona Imaging <http://www.alicona.com/>`_

Support
-------

SCIFIO: |no|

Export: |no|

Officially Supported Versions:  1.0

Supported Metadata Fields: `Alicona
Reader <http://dev.loci.wisc.edu/trac/software/wiki/AliconaReader>`_

We currently have: 

-  an `AL3D specification
   document
   <http://www.alicona.com/home/fileadmin/alicona/downloads/AL3DFormat.pdf>`_ (v1.0,
   from 2003, in PDF)
-  only one AL3D dataset

We'd like to have: 

-  more AL3D datasets (Z series, T series, 16-bit)

Ratings
-------

Pixels: |Very Good|

Metadata: |Very Good|

Openness: |Very Good|

Presence: |Poor|

Utility: |Good|

Additional Information
----------------------

Source Code: :source:`Alicona
Reader <components/bio-formats/src/loci/formats/in/AliconaReader.java>`

Notes: 

Known deficiencies:

-  Support for 16-bit AL3D images is present, but has never been tested.
-  Texture data is currently ignored.
