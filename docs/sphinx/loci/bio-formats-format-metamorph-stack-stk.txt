MetaMorph Stack (STK)
=====================

Extensions:  .stk, .nd

Support
-------

SCIFIO: |no|

Export: |no|

Supported Metadata Fields: `Metamorph Reader <http://dev.loci.wisc.edu/trac/software/wiki/MetamorphReader>`_

We currently have: 

-  An STK specification document (from 2006 Nov 21, in DOC)
-  An older STK specification document (from 2005 Mar 25, in DOC)
-  An ND specification document (from 2002 Jan 24, in PDF)
-  A large number of STK datasets

Ratings
-------

Metadata: |Very Good|

Openness: |Very Good|

Presence: |Very Good|

Utility: |Good|
