PNG (Portable Network Graphics)
===============================

Extensions: .png

Developer: `PNG Development Group <http://www.libpng.org/pub/png/pngnews.html>`_

Support
-------

SCIFIO: |yes|

Export: |yes|

Supported Metadata Fields: `APNG Reader <http://dev.loci.wisc.edu/trac/software/wiki/APNGReader>`_

Freely Available Software: `PNG Writer plugin for
ImageJ <http://rsb.info.nih.gov/ij/plugins/png-writer.html>`_

We currently have: 

-  A `PNG specification
   document <http://www.libpng.org/pub/png/spec/iso/>`_ (W3C/ISO/IEC
   version, from 2003 Nov 10, in HTML)
-  Several PNG datasets

Ratings
-------

Pixels: |Very Good|

Metadata: |Good|

Openness: |Outstanding|

Presence: |Outstanding|

Utility: |Poor|

Additional Information
----------------------

Source Code: :source:`APNG Reader <components/scifio/src/loci/formats/in/APNGReader.java>`

Notes: 

-  Bio-Formats can save individual planes as PNG.
-  Bio-Formats uses the `Java Image
   I/O <http://java.sun.com/j2se/1.4.2/docs/guide/imageio/index.html>`_
   API to read and write PNG files.
