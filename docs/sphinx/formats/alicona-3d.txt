Alicona 3D
===============================================================================

Extensions: .al3d


Owner: `Alicona Imaging <http://www.alicona.com/>`_

**Support**


BSD-licensed: |no|

Export: |no|

Officially Supported Versions: 1.0

Supported Metadata Fields: :doc:`Alicona 3D <alicona-3d-metadata>`



We currently have:

* an `AL3D specification document <http://www.alicona.com/home/fileadmin/alicona/downloads/AL3DFormat.pdf>`_ (v1.0, from 2003, in PDF) 
* a few AL3D datasets

We would like to have:

* more AL3D datasets (Z series, T series, 16-bit)

**Ratings**

Presence: |Poor|

Utility: |Good|

Source Code: :bfreader:`AliconaReader.java`

Known deficiencies: 

- Support for 16-bit AL3D images is present, but has never been tested. 
- Texture data is currently ignored.
