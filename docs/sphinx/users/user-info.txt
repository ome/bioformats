Installation
============

Download one of the files from `the Bio-Formats downloads page <http://www.openmicroscopy.org/site/products/bio-formats/bio-formats-downloads>`_ and place it in an appropriate directory. Specific installation instructions
depend upon the application you are using - see below.

SCIFIO
======

SCIFIO provides the core architecture of the Bio-Formats library and also includes reader and writer implementations for open file formats. The more permissive BSD license enables non-GPL third party software to read and write OME-TIFF using SCIFIO alone. For more information see:

.. toctree::
    :maxdepth: 1
    :titlesonly:

    scifio
